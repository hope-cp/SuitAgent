# Skills 目录

## 用途说明

该目录存放SuitAgent系统使用的**Claude Code技能（Skills）**配置。Skills是Claude Code提供的内置能力扩展，可以通过专门的工具调用执行特定功能。

## 技能列表

当前已集成的技能：

| 技能名称 | 目录 | 功能说明 |
|----------|------|----------|
| **xlsx** | [xlsx/](xlsx/) | 电子表格处理 - 创建、编辑、分析.xlsx文件 |
| **pdf** | [pdf/](pdf/) | PDF文档处理 - 提取、合并、分割、OCR识别 |
| **docx** | [docx/](docx/) | Word文档处理 - 创建、编辑、格式转换 |
| **pptx** | [pptx/](pptx/) | PowerPoint处理 - 演示文稿制作 |
| **md2word** | [md2word/](md2word/) | Markdown转Word转换工具 |
| **skill-creator** | [skill-creator/](skill-creator/) | 技能创建工具 |

## 技能使用方式

### 在Agent中使用
```python
# 在Agent配置中引用
"allowed-tools": ["xlsx", "pdf", "docx"]
```

### 在Command中使用
```markdown
## 技术实现

使用以下技能：
- [xlsx] - 处理电子表格
- [pdf] - 处理PDF文档
```

## 技能详细说明

### 1. xlsx - 电子表格技能
**功能**：处理Excel和CSV文件
- 创建和编辑工作表
- 数据分析和可视化
- 公式计算和函数应用

**典型用途**：
- 工时统计报表
- 案件数据分析
- 期限管理表

### 2. pdf - PDF文档技能
**功能**：PDF文档处理和操作
- 文字提取和OCR识别
- 文档合并和分割
- 页面操作和格式转换

**典型用途**：
- 证据文档解析
- 判决书提取
- 批量PDF处理

### 3. docx - Word文档技能
**功能**：Microsoft Word文档处理
- 创建和编辑.docx文件
- 格式化和样式应用
- 文档模板应用

**典型用途**：
- 法律文书起草
- 委托合同生成
- 批量文档制作

### 4. pptx - 演示文稿技能
**功能**：PowerPoint演示文稿处理
- 创建和编辑演示文稿
- 幻灯片设计和布局
- 图表和多媒体应用

**典型用途**：
- 案件汇报PPT
- 客户展示材料
- 培训课件制作

### 5. md2word - 转换工具
**功能**：Markdown到Word的格式转换
- 保持文档结构
- 应用预设格式
- 支持自定义样式

**典型用途**：
- 法律文书格式转换
- 报告格式规范化
- 批量文档格式转换

### 6. skill-creator - 工具创建
**功能**：创建自定义技能
- 技能脚手架生成
- 配置模板提供
- 最佳实践指导

**典型用途**：
- 开发新技能
- 技能模板创建
- 技能文档编写

## 添加新技能

### 方法1：使用现有技能
1. 检查是否已有相关技能
2. 在Agent或Command中引用
3. 编写使用说明

### 方法2：创建自定义技能
参考[skill-creator/](skill-creator/)目录：
1. 使用技能创建工具
2. 编写技能代码
3. 配置技能文档
4. 测试和验证

## 最佳实践

1. **优先使用内置技能**：内置技能经过充分测试，稳定可靠
2. **正确引用技能**：在Agent的`allowed-tools`中明确列出
3. **编写使用说明**：每个技能使用都应有清晰说明
4. **避免重复功能**：确认技能功能不与现有工具重复
5. **保持文档更新**：技能更新时同步更新文档

## 注意事项

- ❌ **不要**在Skills目录放置非技能文件
- ❌ **不要**修改内置技能的配置文件
- ❌ **不要**将Agent工具函数放在Skills目录
- ✅ **必须**遵循每个技能的文档规范
- ✅ **应该**在代码中正确引用技能
- ✅ **可以**为自定义技能编写说明文档

## 与其他模块的关系

- **与 `agents/`**：Agent通过`allowed-tools`引用技能
- **与 `commands/`**：Command说明中列出使用的技能
- **与 `tools/`**：工具函数与技能功能互补
- **与 `memory/`**：技能使用说明可放在Memory中

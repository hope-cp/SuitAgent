# 🤝 01 - 委托材料

## 功能定位
从委托洽谈开始到正式签订委托的所有材料

## 关联Agent
**Writer** - 生成委托文书（新增第13-15种文书类型）

## 包含内容

### 委托合同/
- `委托代理合同.md` - 委托代理合同模板
- `委托合同-公司版.md` - 公司委托合同
- `委托合同-个人版.md` - 个人委托合同

### 授权书/
- `授权委托书-公司.docx` - 公司授权委托书（Word格式）
- `授权委托书-个人.docx` - 个人授权委托书（Word格式）
- `特别授权委托书.docx` - 特别授权委托书

### 谈话笔录/
- `谈话笔录.docx` - 客户沟通记录（Word格式）
- `初次谈话记录.md` - 初次会见记录
- `重要谈话纪要.md` - 关键决策性谈话

### 服务质量监督卡/
- `服务质量监督卡.docx` - 服务质量承诺（Word格式）
- `服务质量标准.md` - 服务质量说明
- `投诉渠道.md` - 客户投诉和反馈渠道

### 送达地址确认书/
- `法律文书送达地址确认书.docx` - 文书送达地址确认（Word格式）
- `电子送达确认书.docx` - 电子送达方式确认

### 初步法律意见书/
- `初步法律意见书.md` - 初步案件分析
- `案件可行性分析.md` - 胜诉可能性评估
- `风险提示.md` - 初步风险识别

### 委托材料清单/
- `委托材料清单.md` - 材料索引
- `材料接收确认.md` - 客户提交材料确认
- `材料清单更新.md` - 材料补充记录

## 字段映射

### YAML字段 → Word占位符
```yaml
client_name → {client}
lawyer_name → {lawyer}
case_type → {case_type}
委托日期 → {year}-{month}-{day}
```

## 数据流转
- 案件启动时生成
- 被所有Agent引用确认授权范围
- 作为律师执业合规依据

## 使用场景
1. **初次接触** - 制作初步法律意见书
2. **委托确立** - 签署正式委托合同和授权书
3. **服务质量管理** - 监督卡和投诉渠道
4. **材料管理** - 委托材料清单和确认

# 📎 05 - 证据材料

## 功能定位
证据目录、质证意见、证据分析报告的统一管理

## 关联Agent
**EvidenceAnalyzer** - 证据分析和质证意见

## 包含内容

### 证据清单/
- `证据目录.xlsx` - 完整证据目录（EvidenceAnalyzer输出）
- `证据分类清单.md` - 按证据类型分类的清单
- `关键证据清单.md` - 核心关键证据列表
- `辅助证据清单.md` - 辅助性证据列表

### 质证意见/
- `质证意见书.md` - 质证分析报告
- `证据三性分析.md` - 真实性、合法性、关联性分析
- `有利证据分析.md` - 对我方有利的证据分析
- `不利证据分析.md` - 对我不利的证据分析
- `证据证明力评估.md` - 证据的证明力评估

### 证据分类/
- `书证/` - 书面证据
  - `合同协议/`
  - `财务报表/`
  - `往来函件/`
  - `其他书证/`
- `物证/` - 实物证据
- `电子证据/` - 电子数据和电子文件
  - `电子邮件/`
  - `聊天记录/`
  - `网站数据/`
  - `其他电子证据/`
- `证人证言/` - 证人陈述和证言
- `鉴定意见/` - 司法鉴定意见

### 补充证据建议/
- `补充证据建议.md` - 证据缺口分析
- `证据保全申请.md` - 证据保全申请书
- `调取证据申请.md` - 申请法院调取证据
- `鉴定申请.md` - 申请司法鉴定

### 证据保全申请/
- `诉前保全申请书.md` - 起诉前证据保全
- `诉中保全申请书.md` - 诉讼中证据保全
- `保全裁定书.md` - 法院保全裁定
- `保全执行情况.md` - 保全措施执行情况

## 数据流转
- 支持法律研究（03-Researcher）提供法条依据
- 为策略制定提供证据支持（02-Strategist）
- 为文书起草提供证据引用（06-Writer）
- 受对方提交材料影响（07）需动态更新

## 工作流程
```
DocAnalyzer分析证据 → EvidenceAnalyzer深度分析 → 质证意见书
              ↓              ↓                    ↓
          初步分类      三性分析              质证策略
```

## 输出文件示例
- `证据目录_YYYYMMDD.xlsx`
- `质证意见书_YYYYMMDD.md`
- `证据三性分析_YYYYMMDD.md`
- `补充证据建议_YYYYMMDD.md`

## 使用场景
1. **证据收集** - 整理和分类所有证据
2. **质证准备** - 准备质证意见和分析
3. **证据保全** - 申请证据保全措施
4. **补充证据** - 识别证据缺口并补充
5. **庭审准备** - 准备证据展示和质证

# 📅 00 - 日程管理

## 功能定位
案件时间线、法定期限、工时统计、进度管理的统一入口

## 核心文件
- `[案件编号].yaml` - 案件管理看板数据（核心文件）
- `[案件编号].md` - 案件工作记录看板
- `工时记录/` - 详细工时追踪
- `期限提醒/` - 法定期限预警
- `日程安排/` - 案件时间线
- `任务清单/` - 任务管理
- `费用跟踪/` - 费用预算与实际支出

## 关联Agent
**Scheduler** - 负责期限管理、工时统计、时间线更新

## 包含内容模板

### 工时记录/
- `工时记录模板.md` - 标准工时记录格式
- `[YYYY-MM]工时统计.md` - 月度工时汇总
- `工时报表.xlsx` - 工时数据表格

### 期限提醒/
- `法定期限清单.md` - 所有关键期限列表
- `期限预警提醒.md` - 近期到期事项
- `期限管理日历.ics` - 日历格式期限提醒

### 日程安排/
- `案件时间线.md` - 完整案件进展时间线
- `庭审安排.md` - 庭审相关安排
- `重要节点.md` - 案件关键节点提醒

### 任务清单/
- `任务清单.md` - 当前待办任务
- `优先级任务.md` - 高优先级任务列表
- `已完成任务.md` - 任务完成记录

### 费用跟踪/
- `费用预算.md` - 案件费用预算
- `实际支出.md` - 费用实际支出记录
- `费用明细.xlsx` - 费用明细表格

## 数据流转
- 被所有Agent引用，用于状态同步
- 期限数据触发时间相关提醒
- 工时数据支持成本核算

## 使用场景
1. **案件启动** - 创建时间线和初始期限
2. **期限管理** - 跟踪所有法定期限
3. **工时统计** - 记录和统计工作时间
4. **进度汇报** - 生成案件进展报告
5. **费用管理** - 预算控制和支出跟踪

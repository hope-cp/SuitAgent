## 期限管理

### 关键期限
- **应诉期限**：收到起诉状后15日内
- **答辩期限**：收到起诉状后15日内
- **举证期限**：开庭前或法院指定期限
- **上诉期限**：收到判决书后15日内
- **申请执行期限**：判决生效后2年内

### 期限计算规则
```python
截止日期 = 开始日期 + 法定期限天数
剩余天数 = 截止日期 - 当前日期
```

## 输出文件示例

- `期限提醒_YYYYMMDD.md`

## 使用场景
1. **期限管理** - 跟踪所有法定期限
2. **开庭准备** - 根据传票准备庭审
3. **判决分析** - 分析判决书内容
4. **上诉决策** - 根据判决结果决定是否上诉
5. **执行准备** - 判决生效后准备执行

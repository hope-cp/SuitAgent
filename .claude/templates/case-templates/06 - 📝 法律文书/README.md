# 📝 06 - 法律文书

## 功能定位
律师起草的所有法律文书，按文书类型分类管理

## 关联Agent
**Writer** - 起草各类法律文书（支持15种文书类型）

## 包含内容

### 起诉状/
- `起诉状.md` - 一审起诉状
- `起诉状附件清单.md` - 起诉状附件清单
- `诉讼请求详述.md` - 诉讼请求的详细说明

### 答辩状/
- `答辩状.md` - 一审答辩状
- `答辩理由.md` - 答辩理由详细说明
- `反驳诉讼请求.md` - 对原告诉讼请求的反驳

### 代理词/
- `一审代理词.md` - 一审庭审代理词
- `二审代理词.md` - 二审庭审代理词
- `代理词要点.md` - 代理词核心要点

### 质证意见书/
- `质证意见书.md` - 对证据的质证意见
- `质证要点.md` - 质证的核心要点
- `证据异议.md` - 对证据的异议意见

### 申请书/
- `财产保全申请书.md` - 申请财产保全
- `证据保全申请书.md` - 申请证据保全
- `鉴定申请书.md` - 申请司法鉴定
- `调取证据申请书.md` - 申请法院调取证据
- `延期举证申请书.md` - 申请延期举证
- `证人出庭申请书.md` - 申请证人出庭

### 上诉状/
- `上诉状.md` - 二审上诉状
- `上诉理由.md` - 上诉理由详细说明
- `新证据清单.md` - 二审新证据清单

### 律师函/
- `律师函.md` - 对外沟通函件
- `催告函.md` - 催告对方履行义务
- `律师函回复.md` - 对律师函的回复

### 调解协议/
- `调解协议草案.md` - 调解协议初稿
- `调解协议最终版.md` - 最终达成的调解协议
- `调解笔录.md` - 调解过程记录

### 法律意见书/
- `专项法律意见书.md` - 针对特定问题的法律意见
- `风险提示函.md` - 法律风险提示

### 其他文书/
- `追加当事人申请书.md` - 追加第三人或共同诉讼人
- `管辖权异议申请书.md` - 对管辖权的异议
- `反诉状.md` - 提起反诉
- `撤回起诉申请书.md` - 申请撤回起诉

## 字段映射

### YAML字段 → Word占位符
```yaml
client_name → {client}
lawyer_name → {lawyer}
opposite_party → {opposite}
court_name → {court}
case_cause → {case_cause}
```

## 工作流程
```
法律研究(03)完成 → Writer基于法理生成文书 → Reviewer质量审查
      ↓                    ↓               ↓
    法理依据            文书起草          质量把关
```

## 输出文件示例
- `起诉状_YYYYMMDD.md`
- `答辩状_YYYYMMDD.md`
- `代理词_YYYYMMDD.md`
- `质证意见书_YYYYMMDD.md`
- `申请书_YYYYMMDD.md`
- `上诉状_YYYYMMDD.md`

## 使用场景
1. **应诉准备** - 起草答辩状和质证意见
2. **庭审准备** - 准备代理词和庭审文书
3. **程序申请** - 申请各类程序性事项
4. **二审上诉** - 起草上诉状
5. **对外沟通** - 发送律师函等

# 📥 07 - 对方提交

## 功能定位
对方当事人及其代理人提交的所有材料

## 关联Agent
**DocAnalyzer** - 分析对方材料
**EvidenceAnalyzer** - 分析对方证据

## 包含内容

### 起诉状/
- `对方起诉状.md` - 对方当事人的起诉状
- `对方诉讼请求.md` - 对方诉讼请求分析
- `对方事实陈述.md` - 对方陈述的事实
- `对方起诉状附件.md` - 起诉状附件清单

### 答辩书/
- `对方答辩书.md` - 对方在二审阶段的答辩
- `对方答辩理由.md` - 对方答辩理由分析
- `对方反驳观点.md` - 对方反驳我方观点

### 反驳证据/
- `对方反驳证据清单.md` - 对方提供的反驳证据
- `对方证据分析.md` - 对对方证据的分析
- `对方证据目录.xlsx` - 对方证据目录表
- `反驳我方证据.md` - 对方对我方证据的反驳

### 质证意见/
- `对方质证意见.md` - 对方对我方证据的质证意见
- `对方三性异议.md` - 对方对证据三性的异议
- `对方证据质疑.md` - 对方对我方证据的质疑

### 申请书/
- `对方申请书汇总.md` - 对方提交的各类申请
- `对方保全申请.md` - 对方申请的保全措施
- `对方鉴定申请.md` - 对方申请的鉴定
- `对方延期申请.md` - 对方申请的延期

### 上诉材料/
- `对方上诉状.md` - 对方当事人的上诉状
- `对方上诉理由.md` - 对方上诉理由分析
- `对方二审新证据.md` - 对方在二审提交的新证据

### 和解方案/
- `对方和解意向书.md` - 对方提出的和解意向
- `对方和解条件.md` - 对方和解的具体条件
- `和解谈判记录.md` - 和解谈判过程记录

## 数据流转
- 为争议焦点更新提供参考（03-IssueIdentifier）
- 为策略调整提供依据（02-Strategist）
- 为法律研究提供对方观点（03-Researcher）
- 为质证策略提供参考（05-EvidenceAnalyzer）

## 工作流程
```
对方提交材料 → DocAnalyzer分析 → 更新争议焦点 → 调整策略
       ↓              ↓            ↓          ↓
    新材料识别    要素提取      焦点更新     应对方案
```

## 输出文件示例
- `对方起诉状分析_YYYYMMDD.md`
- `对方答辩书分析_YYYYMMDD.md`
- `对方反驳证据分析_YYYYMMDD.md`
- `对方质证意见分析_YYYYMMDD.md`
- `对方申请书分析_YYYYMMDD.md`
- `对方上诉材料分析_YYYYMMDD.md`

## 使用场景
1. **应诉准备** - 分析对方起诉状和证据
2. **质证准备** - 分析对方质证意见
3. **策略调整** - 根据对方材料调整策略
4. **二审准备** - 分析对方上诉材料
5. **和解谈判** - 了解对方和解意向
